# 制作アプリのタイトル：ラノベならべ さくらサーバー利用

## 制作アプリの説明
- ライトノベルをタイトルで検索して探せます
- その中から自分の好きな作品をお気に入りに保存できます
- 保存した作品の、特に好きなところを保存することができます
- ユーザーごとの傾向、本ごとの傾向を比較し、マッチの評価値を計算しておすすめ表示します

## 工夫した点・こだわった点
- RDB化することができました
- 今までお気に入りポイント保存でgood_points_tableに本の情報を全部入れていたり、情報取得時にisbn基準で探していたものを中間テーブルを作って本のidと、ユーザーのidで探すようになりました

## 次回トライしたこと（または機能）
- 見た目や動作の改善は進みませんでした
- アカウント作成画面とトップ画面に情報を増やしたいです
