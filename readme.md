# 制作アプリのタイトル：ラノベならべ さくらサーバー利用

## 制作アプリの説明
- ライトノベルをタイトルで検索して探せます
- その中から自分の好きな作品をお気に入りに保存できます
- 保存した作品の、特に好きなところを保存することができます
- ユーザーごとの傾向、本ごとの傾向を比較し、マッチの評価値を計算しておすすめ表示します

## 工夫した点・こだわった点
- ログイン機能実装し、パスワードの更新も行えるようにしました 特にヘッダーに折りたたんでログイン状態がわかったり、シンプルにログインするだけの時に画面遷移が起こらないようにすることなどを考えて作りました
- UI/UXを意識して操作性を高めました
- 合わせて全体的なデザインとレイアウトを修正してとにかく見やすくなるようにしました
- お気に入りポイント保存をワンタッチにするにあたり、冪等にすることをDB側、フロント側両方で実装しました

## 次回トライしたこと（または機能）
- ログイン周りは機能実装しただけでサービスの一部と認識されるようなデザインにしたいです
- 本の検索周りがあいまい検索が行えない状態なのと、コンセプト的にシリーズ物はシリーズの評価として考えたいのでそこをうまく処理できるようにしたいです
